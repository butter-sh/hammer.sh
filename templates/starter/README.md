## Configuration

Edit `arty.yml` to configure your project:

- **name**: Project name
- **version**: Current version
- **description**: Project description
- **references**: Dependencies (other arty.sh projects)
- **main**: Entry point script
- **scripts**: Available commands

## Development

### Adding Dependencies

Add dependencies to the `references` section in `arty.yml`:

```yaml
references:
  - https://github.com/username/bash-utils.git
  - https://github.com/username/logger.git
```

Then install them:

```bash
arty deps
```

### Using Dependencies

In your bash scripts:

```bash
#!/usr/bin/env bash

# Source dependencies
source <(arty source bash-utils)

# Use functions from dependencies
utils_function
```
