## ✨ New Template Features Documented

### Semantic Versioning
- Major, minor, patch bumping

- Git tag creation

- Updates CHANGELOG.md file
- Integrated into release workflow

### Git Hooks
- Pre-commit validation
- bash -n syntax checking
- shellcheck integration (optional)
- Pluggable hook system

### Monorepo Support
- Batch operations on multiple projects
- Glob pattern filtering
- Individual error handling
- Project discovery
